# 🏎️ Motorsport World

## Описание проекта

**Motorsport World** — это информационный веб-сайт, посвящённый миру автоспорта. Проект знакомит пользователей с популярными гоночными дисциплинами, известными гонщиками, легендарными командами, знаменитыми трассами и календарём соревнований. Сайт выполнен в едином современном стиле с адаптивной вёрсткой и удобной навигацией между страницами.

---

## Список функций

* Главная страница с информацией о проекте.
* Раздел **Sports** с описанием популярных видов автоспорта.
* Раздел **Drivers** с карточками гонщиков, блоком **Driver Spotlight**, цитатами легенд автоспорта и таблицей рекордов.
* Раздел **Teams** с информацией о самых известных командах.
* Раздел **Schedule** с календарём гонок.
* Раздел **Circuits** с описанием легендарных трасс, их характеристиками и интересными фактами.
* Единый дизайн всех страниц.
* Адаптивная вёрстка для компьютеров, планшетов и мобильных устройств.

---

## Используемые технологии

* HTML5

## Инструкция по запуску

1. Скачать или клонировать репозиторий с GitHub.
2. Открыть папку проекта.
3. Запустить файл **index.html** в любом современном веб-браузере (Google Chrome, Microsoft Edge и т.д).

Дополнительная установка программ или библиотек не требуется.
